# Prueba Técnica Zweicom - Laura Merino

## Requisitos

- Python 3.0 o superior
- Flask 2.1 o superior

## Instalación

- `python -m pip install Flask`

## Ejecución App

- `python -m flask run`
- El endpoint es `http://127.0.0.1:5000/fibonacci/<numero>`, donde `<numero>` es la posición n-ésimo término de la sucesión de Fibonacci

## Ejecución Tests Unitarios

- `python -m unittest tests.py`
